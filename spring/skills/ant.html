# Ant路径风格

  • /WEB-INF/*.xml ：匹配 /WEB-INF 目录下的任意 xml 文件
  • /WEB-INF/**/beans-*.xml ：匹配 /WEB-INF 下面任意层级目录的 beans- 开头的 xml 文件
  • /**/*.xml ：匹配任意 xml 文件
